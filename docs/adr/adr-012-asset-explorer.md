# ADR-012: Asset Explorer

> **Implementation status (2026-07-09):** Implemented.
> Asset Explorer pages exist for systems/apps, versions, update status, release links, related access events, and asset metadata/actions.



In addition to the IP Explorer, there should already be another core object for V1:

`Asset Explorer`

This lets you click an app, for example:

`Home Assistant`

and see:

```none
Version
Update available
Router
Accesses
Errors
```

or:

`Nextcloud`

with:

```none
404
500
Geoblocks
Top URLs
```
