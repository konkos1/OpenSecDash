# ADR-005: Insights

> **Implementation status (2026-07-09):** Implemented.
> Insights are stored separately from raw events and generated by rule-based/declarative logic without replacing original event data.



OpenSecDash should not only display data.

It should detect relationships.


---

Example:

```none
Access Event
↓
404
```

plus

`Geoblock Event`

results in:

`404 probably caused by geoblock`


---

**Insights never replace original data.**

They are generated additionally.
